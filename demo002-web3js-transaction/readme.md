### 2. 工程初始化

```
mkdir test
cd test
npm init -y
npm install web3 solc@0.8.17 fs dotenv @openzeppelin/contracts

编写代码
vim xxx.sol
vim xxx.js

```

### 3. 执行部署

```

配置 eth 私钥
export ETH_PRIVATE_KEY=0x77732d9f821695f3d4644e4b5f9d2528bf2a93c9a5b8733a6cdbb2c56f18c6ad

执行部署
node xxxx.js
```
